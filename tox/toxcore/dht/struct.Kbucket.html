pub struct Kbucket { /* fields omitted */ }

K-buckets structure to hold up to KBUCKET_MAX_ENTRIES * BUCKET_DEFAULT_SIZE nodes close to own PK.

Nodes in bucket are sorted by closeness to the PK; closest node is the first, while furthest is last.

Further reading: Tox spec.

Add PackedNode to Kbucket.

Node can be added only if:

  • its kbucket index is lower than the number of buckets.

  • Bucket to which it is added has free space or added node is closer to the PK than other node in the bucket.

    Returns true if node was added successfully, false otherwise.
